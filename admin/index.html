Bóveda cifrada local para documentos y servicios.

La web solo carga la herramienta. Los datos viajan contigo en un archivo .bmbvault, por ejemplo en un USB, y se descifran en este navegador con tu frase de paso.

Sin servidor

Los datos no se envían a BMB, GitHub ni Arsys.

Cifrado

AES-GCM con clave derivada por PBKDF2 en tu navegador.

USB

Exporta la bóveda y guárdala fuera de la web pública.

Sistema soberano

Lo que vendemos debe funcionar primero para nosotros.

Aquí practicamos nuestro propio modelo: catálogo de servicios, borradores, propuestas, trazabilidad y papel corporativo, sin exponer datos sensibles en la web pública.

Puedes guardar una copia cifrada en USB y enviarla manualmente por correo como adjunto. El correo solo debe transportar la bóveda cifrada, nunca documentos en claro ni la frase de paso.
