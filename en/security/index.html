Security

Security and Trust

Last updated: June 15, 2026.

BMB Digital Labs designs with a privacy-by-design approach, exposure minimization and data control. Absolute security is not promised: the specific measures depend on each product, architecture, provider and real configuration.

Measures planned according to product

  • Access control.
  • Temporary links.
  • Traceability and activity records.
  • Encryption when implemented and documented.
  • Data export and deletion.
  • Backups.
  • Activity audit.

Incident channel

If you detect a vulnerability or security incident, contact: gpmx@gmx.com.

Please include a clear description, reproduction steps, affected URL and any useful technical evidence. Do not access, modify, delete or extract third-party data.
