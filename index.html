Vision

Menos dependencia. Mas control.

El producto principal no es una web que lo guarda todo en un servidor. Es una oficina digital que trabaja con archivos del usuario, exportables, cifrados y portables.

Funcionamiento

Una oficina virtual sin entregar tu archivo maestro.

El usuario conserva sus datos. La aplicación solo abre, descifra, edita y vuelve a guardar cuando el usuario lo decide.

La idea es sencilla: cargar archivo, trabajar, exportar, cerrar sesión, limpiar el navegador y continuar teniendo una copia propia fuera de la plataforma.

Cuando haga falta servidor, debe ser mínimo: publicación, sincronización opcional, cobro o respaldo cifrado. No como almacén obligatorio de la vida privada del cliente.

Productos previstos Principios de privacidad

Flujo base

Productos pensados para datos sensibles, documentos, negocios pequeños y usuarios que no quieren depender de una nube cerrada.

1. Abrir
El usuario sube su archivo JSON o bóveda cifrada desde su equipo.
2. Trabajar
La oficina virtual permite editar, ordenar, generar documentos e informes.
3. Guardar
Se exporta una nueva copia cifrada o firmada, lista para USB, disco o respaldo privado.
4. Salir
Se bloquea la sesión, se limpia el navegador y no queda una base de datos central expuesta.

Privacidad práctica: menos promesas, más control real sobre el archivo.
