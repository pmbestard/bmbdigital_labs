Datos que se recogen

El formulario de contacto puede recoger nombre, correo electrónico, tipo de proyecto, mensaje y confirmación de aceptación de la política de privacidad. Además, el alojamiento web puede tratar datos técnicos básicos como dirección IP, agente de usuario, fecha y hora de acceso, URL solicitada y registros de seguridad necesarios para operar el sitio.

Finalidades y base jurídica

Los datos se usan para responder a la consulta, entender el contexto del proyecto, preparar una propuesta si procede y mantener una comunicación razonable vinculada a esa solicitud. La base jurídica principal es el consentimiento. En fases posteriores, si existe contratación, podrían aplicarse ejecución contractual y obligaciones legales, que deberán documentarse antes de prestar servicios continuados.

Proveedores y transferencias

El alojamiento público detectado es GitHub Pages. El formulario actual prepara un correo mediante el cliente de email del usuario y no envía datos a una plataforma de formularios desde el navegador. Si se incorpora un proveedor externo, deberá identificarse antes de su uso.

No se han detectado Google Analytics, Meta Pixel, Hotjar, Plausible, Umami, YouTube embeds, Google Maps ni scripts externos de analítica/publicidad en la web principal. Si en el futuro se incorporan proveedores con transferencias internacionales, se informará y se aplicarán las garantías correspondientes.

Conservación

Las consultas se conservarán durante el tiempo necesario para responder, hacer seguimiento razonable y, si procede, preparar una propuesta. Cuando los datos dejen de ser necesarios, deberán eliminarse o archivarse conforme a criterios de minimización y obligaciones aplicables.

Derechos

Puede solicitar acceso, rectificación, supresión, oposición, limitación, portabilidad y retirada del consentimiento escribiendo a gpmx@gmx.com. También puede reclamar ante la Agencia Española de Protección de Datos: www.aepd.es.

Medidas de seguridad

BMB Digital Labs trabaja con enfoque de privacidad por diseño, control de datos y medidas técnicas y organizativas razonables. No se promete seguridad absoluta, privacidad total garantizada ni cifrado extremo a extremo salvo que esté implementado y documentado en un producto concreto.
